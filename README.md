# Proyecto Final Entornos de Desarrollo

## Sistema de Venta de Tickets para Parque de Atracciones

Aplicación de escritorio desarrollada en Java utilizando el patrón MVC, Swing y MySQL.

## Tecnologías utilizadas

- Java
- Swing
- JDBC
- MySQL
- MVC
- GitHub
- JavaDoc

## Funcionalidades

- Consulta de eventos
- Compra de tickets
- Confirmación de compra
- Registro de ventas en MySQL
- Gestión mediante arquitectura MVC

## Estructura del proyecto

- modelo
- dao
- vista
- controlador
- main

## Base de datos

La aplicación utiliza MySQL y JDBC para conectarse a la base de datos `venta_tickets_parque`.
